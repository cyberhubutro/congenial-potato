git all commands

  1. git clone [url] - для клонирования репозитория
  2. cd [path] - для перемещения из папки в другую папку в терминале
  3. git add [file_path] - для добавления файла в индексацию
  4. git commit - для фиксации наших изменений
  5. git push [remote] - для того что бы наши изменения отправить в remote
